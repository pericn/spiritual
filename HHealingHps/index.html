📱 安卓设备指南

操作步骤：

1. 浏览器打开页面 → 点击下载
2. 在文件管理中打开 .ics 文件
3. 选择「用日历应用打开」 4. 根据提示完成订阅（如果需要日历提醒，请检查订阅页面中是否有类似「移除提醒」的设置并确保其关闭）

 苹果设备指南

操作步骤：

1. 打开「日历」应用 → 点击页面正下方的「日历」文字
2. 选择↙️左下角「添加订阅日历」
3. 粘贴复制的链接，并点击「订阅」按钮 4. 在弹出页面中的最底部，点击「订阅详细信息」，关闭「移除提醒」开关 5. 返回，并点击右上角「添加」
