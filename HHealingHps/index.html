📱 安卓设备操作指南

  1. 使用浏览器打开本页面 → 点击下载按钮
  2. 在手机文件管理器中找到下载的 .ics 文件
  3. 长按文件 → 选择「用日历应用打开」
  4. 根据提示完成订阅
    如需接收提醒：在日历应用的订阅设置中
    确认已开启「提醒通知」功能

 苹果设备操作指南

  1. 打开「日历」应用 → 点击底部「日历」标签
  2. 选择左下角「添加订阅日历」选项
  3. 粘贴复制的链接 → 点击「订阅」按钮
  4. 在订阅页面底部点击「订阅详细信息」
  5. 关闭「移除提醒」开关 → 返回点击「添加」
重要提示：必须关闭「移除提醒」开关
才能正常接收课程提醒通知
